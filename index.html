This repository contains code samples and analyses that I've perform as I expand my analytics and data science skillset. All of these samples are written in R and include a bit of enterprise data analysis and some model building. This repository contains:

    Model to predict
    Wine Quality Rating in R         		I used white and red wine data to create a model that predicts the quality rating from chemical attributes of the wine. The model was built using random forest algorithm. The data contains 11 chemical measurements as predictors and a categorical predictor - quality.
